Sobre: Programa que encontra emails em um arquivo de texto. O usuário
insere o diretório de um arquivo de texto, e o programa printa os
emails presentes nele, caso haja algum.

Bibliotecas usadas: os (Apenas para questões gráficas), re

Guia das bibliotecas:
re -> https://docs.python.org/3/library/re.html
os -> https://docs.python.org/3/library/os.html

Observações: Esse projeto foi feito utilizando conhecimentos sobre funções para facilitar a realização.
Caso você ainda não tenha estudado sobre isso, porém, ainda é possível realizá-lo tranquilamente!
Basta escrever o código em um único arquivo normalmente.
